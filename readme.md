## Daji 工具说明
妲己（Daji）是一个命令行工具集，提供多种实用功能来提升开发和日常工作效率。

## 功能列表
* **Host管理** - 快速切换和管理本地host配置
* **Git配置** - Git账户管理和快速切换
* **IP查询** - 查看本机IP地址信息
* **iOS模拟器** - 快速启动iOS模拟器并打开指定URL
* **日期时间处理** - 时间戳与日期格式转换，支持多种日期格式解析
* **密码生成器** - 生成安全随机密码，可自定义字符类型和长度
* **文件批量重命名** - 支持添加前缀、序号和过滤文件类型
* **路径管理** - 查看和管理当前路径信息
* **工具配置管理** - 工具设置的导入导出和备份操作
